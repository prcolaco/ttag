Installation

ttag

Install with npm

npm install --save ttag

Install with yarn

yarn add ttag

ttag-cli

For the whole translation process you also need to install ttag-cli.

Install with npm

npm install --save-dev ttag-cli

Install with yarn

yarn add ttag-cli --dev
